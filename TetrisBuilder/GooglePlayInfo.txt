How to Play:
Main games task place blocks one on others and build Highest Tower or out all blocks.
	1. You can Move tetris's Block by dragging it.
	2. You can Move Cam by dragging background.
	3. You can Rotate Block by screen buttons ↺ and ↻.
	4. You can Zoom the field by screen buttons + and -.
	5. To put block you nedd to release block in touch with others.

The game provides Standert Mode and Endless Mode.
